NextGIS Frontend - набор инструментов для эффективной разработки веб-приложений на базе NextGIS Web

Наша Веб ГИС NextGIS Web, о которой мы подробно рассказывали на предыдущих вебинарах, это не только готовое решение для управления геоданными в сети, но и мощный фундамент для построения ваших собственных веб-приложений, ориентированных на работу с пространственными данными. Для того, чтобы помочь вам быстро и эффективно начать разработку, мы сделали NextGIS Frontend - специальный набор js-библиотек, радикально упрощающий взаимодействие с NGW.

На вебинаре детально познакомимся с NextGIS Frontend (NGF) и обсудим следующие темы:

  • Устройство системы пакетов в NGF
  • Установка библиотек
  • Работа с code.nextgis.com
  • Демонстрация примеров работы с картами и ресурсами NextGIS Web
  • Стилизация, фильтрация, идентификация векторных объектов
  • Прямые запросы через ngw-connect
