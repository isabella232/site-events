NextGIS Web Cloud: dive into the details

NextGIS Web — server-based Web GIS, designed for storing and organizing multi-user access to geodata and creating web maps. This is our flagship product, around which the entire platform is built.

At the first webinar, we met with nextgis.com, figured out what it is, how to register and get started. Details and materials of the introductory webinar are available here: https://nextgis.ru/blog/event-ngw/. In the next webinar, we will continue working with NextGIS Web in the cloud and dive into some details. We plan to tell you about:

  • Setting up vector layers: field aliases, name attributes, hiding attributes
  • Catalog of substrates and adding them to web maps
  • Setting up and working with web maps: managing bookmarks, setting up previews for social networks, editing vector data, and creating annotations
  • Learn more about embedding web maps in third-party sites
  • Learn more about configuring access rights to Web GIS resources

The webinar will take place on March 30, 2021, at 11:00 (msk) on the Zoom platform. To get a link to a meeting, register!
