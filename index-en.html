«Getting started with NextGIS Web Cloud. From registartion to map collection»


NextGIS Web — server-based Web GIS, designed for storing and organizing multi-user access to geodata and creating web maps. This is our flagship product, around which the entire platform is built.

Cloud service nextgis.com is the easiest and most affordable way to get started with NextGIS Web, with it you can create your own full-fledged Web GIS in a few clicks. At the webinar, we will get acquainted with how our cloud works and how to start working with it as comfortably as possible:
  • How to register at nextgis.com
  • Add different data to it - vector and raster, as well as styles for them
  • Publish several maps based on this data and configure them
  • Touch on the mechanism for configuring access rights
